🏠 Hostel Smart Assistant – Complaint & Leave Management System

Mini Project | B.Tech (IT) | Dronacharya Group of Institutions

---

📘 Project Overview

Hostel Smart Assistant is a web-based application designed to simplify hostel management by providing a digital platform for complaint handling and leave management.

It allows students to submit complaints and apply for leave, while the admin can efficiently manage and resolve them through a centralized system.

The system improves communication, reduces manual work, and provides a structured way to track issues and requests.

✔ Secure authentication for users and admin

✔ Leave application system

✔ Status update (Pending / Resolved)
✔ Simple and user-friendly interface
✔ Database integration using SQLite

1. Install Dependencies
   pip install -r requirements.txt

2. Run the Application
   python main.py

3. Open in Browser
   http://127.0.0.1:5000
